JUMP
====

# Instalação

  1. Instale o Pygame e o Git:

    <code>
        apt-get install python-pygame git
    </code>
   

  2. Baixar e Executar o Jogo:

    <code>
        git clone https://github.com/CharlesGarrocho/jump.git
    </code>

    <code>
        cd jump
    </code>

    <code>
        python start.py
    </code>
